# HooBank

A fast & modern landing page for a banking website.

## Technologies

- Preact
- TailwindCSS
- Vite (Build Tool)

## Getting started

- Clone the repository

```sh
$ git clone https://github.com/Dev-Siri/Hoo-Bank.git
```

- Install the dependencies.

```sh
$ pnpm i
```

- Run the local dev server

```
$ pnpm dev
```
